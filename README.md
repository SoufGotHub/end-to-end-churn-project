# Churn Prediction Project

This project aims to predict customer churn using a logistic regression model.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repository.git
   ```
2. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Data

The `data/raw` directory contains the following datasets:

- `Telco-Customer-Churn.csv`: The original dataset used for training the churn prediction model.
- `dataset_churn_simule.csv`: A simulated dataset generated by the `01_generate_dataset.py` script.

## Scripts

The `scripts` directory contains the following Python scripts:

- `01_generate_dataset.py`: This script generates a simulated dataset for churn prediction.
- `02_train_model.py`: This script trains a logistic regression model on the preprocessed data and saves the trained model to the `models` directory.

## Models

The `models` directory contains the trained model:

- `churn_logistic_regression.joblib`: A logistic regression model trained to predict customer churn. This model is generated by the `02_train_model.py` script.

## Docker

The `Dockerfile` in the root directory can be used to build a Docker image for this project.

To build the image, run the following command:

```bash
docker build -t churn-prediction .
```

To run the container, use the following command:

```bash
docker run -it churn-prediction
```

## SQL

The `sql` directory contains the following SQL scripts:

- `schema.sql`: This script defines the database schema for the project.
- `query_extraction_features.sql`: This script contains queries for extracting features from the database.
